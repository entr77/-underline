모든 작업은 확인/컨펌 없이 바로 진행한다.

**Why:** 사용자가 직접 지시한 사항으로, 불필요한 확인 요청 없이 빠르게 실행하길 원함.

**How to apply:**
- "~할까요?", "~진행할게요?", "~하겠습니다" 같은 확인 문구 없이 즉시 실행
- 멀티스텝 작업도 중간에 멈추지 않고 끝까지 완료 후 결과만 보고
- 작업 중 새 메시지가 와도 현재 작업 완료 후 처리
- 단, 되돌릴 수 없는 파괴적 작업(파일 영구 삭제, DB 드롭, force push 등)만 예외
