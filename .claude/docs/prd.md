## 서비스 개요
책을 읽으면서 밑줄 친 문장을 기록하고 다른 사람들과 공유하는 소셜 독서 플랫폼.
사진 한 장으로 책 정보, 페이지, 밑줄 친 문장을 자동으로 인식하여 기록한다.

## 핵심 가치
- **기록**: 사진 한 장으로 밑줄 친 문장을 손쉽게 저장
- **공유**: 피드로 공유하고 같은 책을 읽은 사람들과 연결
- **발견**: 비슷한 취향의 사람들이 밑줄 친 문장과 책 탐색
- **추천**: 내 밑줄과 프로필을 분석해 책과 문장 추천

## 타겟 유저
- 독서를 즐기며 인상적인 문장을 기록/공유하고 싶은 사람
- 다른 독자들이 어디에 밑줄 그었는지 궁금한 사람
- 나와 비슷한 취향의 독자를 발견하고 싶은 사람

## MVP 기능 (P0)
| 기능 | 설명 |
|------|------|
| 회원가입/로그인 | 이메일 또는 Google |
| 밑줄 추가 | 사진 → OCR + 밑줄 감지 → 확인/수정 → 저장 |
| 책 자동 인식 | 카카오 도서 API로 책 제목/저자 자동 검색 |
| 페이지 번호 감지 | OCR로 자동 추출, 수정 가능 |
| 공개 피드 | 전체 밑줄 타임라인 |
| 좋아요 | 다른 사람의 밑줄에 공감 |
| 마이페이지 | 내 밑줄 모음, 프로필 |
| 책별 밑줄 모음 | 같은 책의 모든 밑줄 |

## 다음 이터레이션 (P1)
| 기능 | 설명 |
|------|------|
| 공감자 프로파일 | 이 문장에 밑줄 그은 사람들의 취향 태그 |
| 프로필 기반 탐색 | 특정 유형의 사람들이 밑줄 친 문장/책 |

## 장기 로드맵 (P2)
- 추천 엔진 — 내 밑줄 + 프로필 기반 책/문장 추천

## 핵심 기술 제약
- 이미지에서 밑줄/형광펜 위치 자동 감지 (Google Cloud Vision API)
- 전체 텍스트 OCR 후 좌표 기반 텍스트 블록 매핑
- 카카오 도서 API는 서버에서만 호출 (키 노출 방지)
