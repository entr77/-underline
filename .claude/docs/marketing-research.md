## SNS 독서 밑줄 공유 문화

### 채널별 현황

**인스타그램 (Bookstagram)**
- `#bookstagram` 게시물: 1억 1,900만 건 이상 (2025년 기준)
- 한국 해시태그: `#북스타그램`, `#책스타그램`, `#독서기록`, `#필사`, `#책귀`, `#책밑줄`
- 콘텐츠 형식: 책 표지 + 밑줄 문장 사진, 손글씨 필사 사진, 책+소품 라이프스타일 플랫레이
- 특이사항: 2025년부터 해시태그 팔로우 폐지 → 알고리즘 추천 중심 전환

**틱톡 (BookTok)**
- `#booktok` 조회수: 2,000억 회 이상 (2025년 누적)
- 한국형: `#북톡`, `#책추천`, `#독서브이로그`
- 콘텐츠 형식: "이 문장 때문에 울었다" 류 감성 쇼트, 트렌딩 오디오 + 책 이미지 텍스트 오버레이
- 바이럴 패턴: 밑줄 문장 하나가 퍼져 절판 책 역주행하는 현상 발생

**유튜브 / 쇼츠**
- 독서 브이로그에 "이 책에서 밑줄 친 문장들" 챕터 구성 보편화
- 책 문구 타이포그래피 쇼츠 급증 (텍스트 + 배경음악)

**트위터/X**
- 책 문장 이미지 카드 첨부 트렌드 지속
- 독서인 네트워크의 서로 추천하는 커뮤니티 문화 강함

---

### 공유 동기 / 심리

1. **정체성 표현 (68%)**: "나는 이런 문장에 밑줄 긋는 사람"이라는 자기 소개
2. **사회적 지위**: 좋은 문장 먼저 발굴 → 통찰력 있는 사람으로 인식
3. **감정 대리 표현**: 말로 설명 못할 감정을 책 문장이 대신 말해줌
4. **자기 동기 부여**: 공개 기록 = 자신에게 하는 공개 약속
5. **텍스트힙 반작용**: 숏폼 피로 → 아날로그 텍스트 회귀. 종이책 읽기 = 힙한 행위

### 인기 콘텐츠 패턴

**가장 공유되는 책**
- 한국: 한강 작가 노벨문학상(2024) 이후 "채식주의자", "소년이 온다" 문장 폭발적 공유
- 글로벌: Fourth Wing(78만 TikTok 언급), It Ends with Us(120만 언급)

**가장 반응 좋은 형식**
- 손으로 밑줄 그어진 실물 책 페이지 클로즈업 사진 (진정성 최강)
- 밀드라이너 파스텔 형광펜이 예쁘게 찍힌 사진
- "이 문장에서 책을 덮었다" 캡션 + 클로즈업

### 트렌드 시그널 (2023~2025)

- **텍스트힙** 신조어 대중화 (2024년 국내 Z세대 대표 트렌드)
- 2024 서울국제도서전 역대 최대 방문객 15만 명
- 한국 전자출판 합산 매출 1조 5,959억 원 (+12.2%, 2024년)
- `#필사` 인스타 게시물 급증 — 손으로 옮겨 쓰는 독서 방식이 SNS 콘텐츠로 자리잡음
- BookTok이 출판 시장을 실질적으로 움직이는 힘으로 확인됨 (2025)

## 책 밑줄 도구 생태계

### 물리적 도구

| 도구 | 특징 | 독서가 선호 이유 |
|------|------|----------------|
| 제브라 밀드라이너 | 파스텔 25색, 듀얼팁, 비침 최소 | 독서 커뮤니티 압도적 1위. 사진 찍었을 때 예쁨 |
| 제브라 옵텍스케어 | 수성 잉크, 책 손상 최소 | 책 보존 중시하는 독서가 선호 |
| 인덱스 탭 | 색별 중요도 구분 | 다회독 시 위치 표시 |
| 0.3~0.5mm 연필 | 지울 수 있음 | 도서관 책, 재판매 예정 책 |

**물리적 도구의 한계**: 기록이 책 안에 갇혀 검색/재활용 불가. 책을 잃어버리면 기록 소실. 공유하려면 사진 찍어서 직접 올려야 함.

### 시장 공백 — 밑줄 서비스 차별점

**종이책 + 소셜 + 무료 = 현재 세상에 없는 조합**

| 공백 | 밑줄의 해결 방식 |
|------|----------------|
| 종이책 밑줄의 디지털화 수단 부재 | 사진 한 장 → OCR로 텍스트 자동 추출 |
| 책 정보 수동 입력 | 사진에서 책 정보 자동 인식 (카카오 API 연동) |
| 하이라이트 앱의 소셜 레이어 부재 | 피드 + 좋아요 + 팔로우 구조 |
| 한국어 책 지원 부재 | 카카오 도서 API + 한글 OCR 특화 |

## 마케터 인사이트

### → Planner에게

**발견**: 독자들이 "사진 찍어서 인스타에 올리는" 행동은 이미 대규모로 존재하지만, 텍스트가 이미지 속에 묻혀 검색·재활용·연결이 불가능한 구조적 마찰이 있다.

**근거**: BookTok 2,000억 뷰 + Bookstagram 1.2억 게시물은 수요가 입증된 시장. Readwise는 전자책 헤비유저를 잡고, 국내 플랫폼은 생태계 내에 가두는 구조. 종이책 + 소셜 + 무료 조합은 명확한 공백.

**기획 제안**:
- 한강 노벨상처럼 문화적 모멘텀 이벤트와 연동하는 큐레이션 기능 검토 (P1)
- Readwise의 스페이스드 리피티션 → "오늘의 내 밑줄" 복습 알림으로 간소화하면 리텐션 장치가 됨 (P1)
- `/book/[id]` 히트맵을 "이 책의 독자들이 가장 많이 밑줄 친 문장"으로 전면에 내세울 것 (즉시 가능, P0)
- 우선순위: P1

### → Copywriter에게

**공략 감정**: 외로움 + 공명 — "이 문장에서 나만 멈춘 게 아니었구나"

**핵심 메시지 방향**:
- "당신이 밑줄 친 문장이, 누군가의 오늘을 설명한다"
- "사진 한 장으로 충분합니다" (행동 장벽 낮음 강조)
- "밑줄은 나를 드러내는 가장 솔직한 방법" (정체성 표현 욕구 활용)
- 아날로그 감성 + 디지털 연결 = 두 세계의 최선으로 포지셔닝

**금지 표현**:
- "독서 기록 앱" — 생산성 앱 오인
- "OCR 기술" — 기능 설명은 마케팅 언어 아님
- "플랫폼" — SNS 감성 희석
- "독서량 관리 / 목표 달성" — 자기계발 앱 뉘앙스 절대 금지

## 콘텐츠 훅 — 바이럴 앵글

### "세상에 사람은 2종류로 나뉩니다"

> 출처: https://m.blog.naver.com/hopestory21/222023808143

**핵심 프레임**: 책을 읽으면서 밑줄을 긋는 사람 vs 아무 표시도 남기지 않는 사람 — 어떤 방식이 더 효과적일까?

**왜 주목해야 하는가**:
- 이 프레임은 밑줄 앱의 타겟을 자연스럽게 정의한다 ("밑줄 긋는 사람"이 우리 유저)
- "효과적일까요?" 질문이 댓글·공유 유도 — 참여형 콘텐츠
- 우리 앱의 포지셔닝: 밑줄 긋는 사람들의 기록을 연결하는 서비스

**→ Copywriter에게**:
- "당신은 어느 쪽인가요?" 양자택일 훅을 인스타/틱톡 첫 화면으로 활용
- 2종류 프레임에서 밑줄 긋는 사람 = 더 나은 독자 포지셔닝이 되면 앱 다운로드 동기 자연스럽게 연결
- CTA 방향: "밑줄 긋는 사람이라면, 이제 그 밑줄들을 저장하세요"

**→ Planner에게**:
- 온보딩에서 "어떤 독자인가요?" 타입 선택 UX 검토 (밑줄형 / 깔끔하게 읽는 형)
- 두 번째 타입(표시 안 하는 사람)도 앱 진입 동기 만들 수 있음: "다른 독자들의 밑줄 발견하기"
