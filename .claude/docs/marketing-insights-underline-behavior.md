# 마케팅 인사이트 — 사람들은 책 어디에 밑줄을 긋는가?

> 작성: 마케터 에이전트 + 5개 리서치 에이전트 병렬 수행
> 날짜: 2026-06-18

---

## 1. 위치별 밑줄 패턴

| 구간 | 패턴 | 이유 |
|------|------|------|
| **앞 (1~20%)** | 밀도 높음 | 저자의 핵심 명제·도발적 선언 집중 |
| **중반 (40~60%)** | 밀도 최저 | 서사·논리 전개에 몰입 — 멈추지 않음 |
| **후반 (75~90%)** | 밀도 다시 급증 | 전체 논리 응축, 감정 클라이맥스 |
| **마지막 / 에필로그** | 단독 공유율 최고 | 책 전체의 기억을 한 문장으로 압축 |

Kindle 최다 하이라이트의 약 60~70%가 **후반 30% 구간**에 집중.

---

## 2. 장르별 차이

| 장르 | 밑줄 빈도 | 소셜 공유율 |
|------|----------|-----------|
| 에세이 | 매우 높음 | **최고** — 인스타그램 지배적 |
| 자기계발 | 높음 | 높음 (캡처·저장 형태) |
| 인문/철학 | 중간 | 중간 (긴 인용문 형태) |
| 소설 | 낮음~중간 | 낮음 (맥락 없이 공유 시 의미 약화) |

**안구 추적 연구 근거 (PMC12641876)**: 에세이·시 텍스트에서 텍스트 복귀율 β=0.53 vs 소설 0.00 — 에세이 독자가 가장 많이 멈추고 다시 읽는다는 것이 실험으로 증명됨.

**한국 시장 특이점**: 2020년대 에세이 장르 강세 (최은영·정세랑·김영하류). 한강 노벨상 이후 문학 붐 지속 중.

---

## 3. 밑줄 유도 문장 5가지 유형

1. **역설·반전 구조** — 기대 반대 방향으로 결론. 뇌가 "다시 읽어야 해" 신호를 보냄
2. **짧은 경구 (15~25자)** — 논증 없이 결론만. 공유하기 쉽고 기억에 남음
3. **"나를 정확히 묘사하는 문장"** — "이걸 내가 쓴 게 아닌가" 느끼는 문장. 에세이 장르 지배
4. **이미지화 가능한 비유** — 추상적 감정을 물리적 이미지로 변환한 순간 멈추게 됨
5. **현상에 이름을 붙이는 문장** — 이름 없던 감정에 언어를 부여. 공유 동기 가장 강함

### Kindle 상위 25개 하이라이트 구조 분석

| 패턴 | 빈도 |
|------|------|
| 역설·대조 | 높음 |
| 취약성 언어 ("not equipped to deal with") | 높음 |
| 보편적 선언 + 개인 고백 | 높음 |
| 아이러니 | 중간 |
| 비율·수치 논리 ("ten times as long") | 중간 |

**충격적 발견**: 상위 25개 중 19개가 수잔 콜린스(헝거게임) 문장 — 문학적 권위보다 감정적 직접성이 압도적으로 이긴다.

**공통 공식**:
```
[보편적 선언 또는 개인 고백] + [감정적 취약성 또는 역설]
최적 길이: 12~28단어 (한국어 기준 약 20~50자)
```

상위 하이라이트 100%가 **선언문(declarative sentence)** — 명령문도, 수사적 질문도 없다.

## 4. 심리적 동기 5계층

```
1위 — 공감: "이 문장이 나다"              → 소셜 공유 동기 1위
2위 — 인식 전환: "이런 관점은 처음이다"   → 지적 허영심 자극
3위 — 아름다운 표현: "이 문장이 좋다"     → 개인 소장 성격 강함
4위 — 실용 조언: "이거 써먹어야지"        → 자기계발 장르 특화
5위 — 감정 방어: "이 문장 뒤에 숨고 싶다" → 슬픔·분노를 대신 표현
```

**핵심**: 좋아요를 가장 많이 받는 밑줄은 공감(1위)이지만, 오래 기억되는 밑줄은 인식 전환(2위). 두 레이어를 동시에 다룰 때 서비스 가치 극대화.

## 5. 밑줄 행동의 과학적 근거

### 기억 효과 (2022 메타분석, Educational Psychology Review)

| 효과 | 수치 |
|------|------|
| 밑줄 → 기억력 향상 | effect size **d = 0.36** |
| 밑줄 → 이해력 향상 | d = 0.20 (불안정) |

→ 밑줄은 이해보다 **기억**에 훨씬 강하게 작용. 독자는 "이해하려고"가 아니라 **"잊지 않으려고"** 긋는다.

### 자기 관련성 효과 (Rogers, Kuiper & Kirker, 1977)

같은 단어를 읽어도 "나와 관련된 방식"으로 처리할 때 기억률이 가장 높다 — 구조적·음운적·의미론적 처리를 모두 앞선다.

→ **밑줄은 단순 메모가 아니라 자기 정체성을 기억에 각인하는 행위.**

### 소셜 밑줄의 효과 (ScienceDirect, Computers in Human Behavior)

그룹이 함께 주석을 달고 읽은 독자가 혼자 읽은 독자보다 이해도 테스트에서 더 높은 점수 기록.

→ "같은 책을 읽은 사람들의 밑줄을 보는 것"이 독해력을 실제로 높이는 행위.

## 6. 한국 독서 시장 데이터 (2025)

- **성인 독서율**: 38.5% (처음으로 40% 이하로 하락)
- **연간 평균 독서량**: 종이책 5.4권 / 전자책 1.4권
- **독서 동기 분포**: 즐거움 20.3% / 자기계발 18.5% / 지식 습득 16.5% / 감정적 위로 15.8%
- **밀리의서재**: 910만 회원
- **경쟁 앱**: 북모리(OCR 독서 노트), 북트리(포토+텍스트), 리더스(소셜 하이라이팅)

**인사이트**: 독서율 하락은 위기가 아니라 기회 — 읽는 사람이 줄면 "진짜 독자"의 밀도가 높아진다. 밑줄 서비스의 타겟은 이 38.5% 안에서도 더 깊이 읽는 상위 독자층.

## 7. 한국 북스타그램 특이점

> **"슬프지만 사실인 문장"이 "영감을 주는 문장"을 이긴다**

서구 인용구 문화는 긍정·영감 중심이지만, 한국 북스타그래머는 **우울함·미해결 감정·침묵**을 담은 문장에 더 강하게 반응.

- 한국 출판사들이 인스타그램용으로 책을 설계 중 — 여백 확대, 짧은 인용 문장, 아름다운 타이포그래피
- 주요 해시태그: `#글귀` `#인생글귀` `#글귀스타그램` `#책스타그램`

## 8. 밑줄 행동의 법칙 3가지 (종합)

1. **선언형 + 취약성** — 조언하지 않고, 느끼는 것을 말한다
2. **자기 관련성** — "나를 묘사하는 문장"이 기억에 가장 오래 남는다
3. **맥락 독립성** — 책을 읽지 않아도 혼자 서있는 문장 (20~50자)

**한국 시장 추가 법칙**: 슬픔·모호함을 해소하지 않는 문장 — "괜찮아질 거야"보다 "그건 아무렇지도 않았다"가 더 많이 공유된다.

## 9. 플래너에게 — 기획 제안

**발견**: 밑줄은 책의 앞(각성)과 끝(여운) 구간에 집중되며, 에세이 장르의 짧은 경구가 소셜 공유를 지배한다. 독자는 공감 확인을 위해 공유하고, 같은 문장에 밑줄 그은 타인을 만날 때 가장 강한 유대감을 느낀다.

**기획 제안**:
- "같은 문장에 밑줄 그은 사람 N명" 기능 — 현재 `/underline/[id]`의 "여기서 같이 멈췄어요" 섹션 강화
- 책 위치 기반 히트맵 고도화 — 앞/중/끝 구간 시각화로 확장
- 에세이 카테고리 우선 큐레이션 — 피드 상단 노출

**우선순위**: P1

## 10. 카피라이터에게 — 캠페인 브리프

**캠페인명**: "이 문장, 나만 밑줄 그은 게 아니었다"

**목표**: 가입 전환 — 책 읽는 사람이라면 한 번쯤 느꼈을 감정을 포착해 첫 진입 동기 부여

**타겟 페르소나**: 25~35세, 에세이·인문서를 즐겨 읽고 인상적인 문장을 메모앱이나 사진으로 혼자 저장해온 사람. 인스타그램에 가끔 인용문을 올리지만 "이런 거 올려도 되나"라는 어색함이 있음.

**공략 감정**: 고독한 공명 — 좋은 문장 앞에서 혼자 멈춰 선 순간의 설렘과, 그 감동을 나눌 사람이 없다는 미묘한 외로움의 공존

**핵심 메시지**: "당신이 밑줄 그은 그 문장, 누군가도 같은 자리에서 멈췄습니다."

**금지 표현**: "독서 습관", "리딩 로그", "공유하세요", "커뮤니티", "따뜻하게", "소셜"

**레퍼런스 톤**: Before Sunrise의 둘만 아는 연결감 · 교보문고 "책은 도끼다" 캠페인의 절제된 여백 · Spotify Wrapped의 개인화 카피

## 11. 위치 데이터 보정 — Hawking Index (Jordan Ellenberg, 2014)

앞서 "후반 집중" 가설을 제시했으나, 실제 Kindle 데이터는 장르에 따라 다르다.

| 책 | Hawking Index | 해석 |
|----|--------------|------|
| 피케티 《21세기 자본》 | **2.4%** | 독자 대부분이 26페이지에서 이탈 |
| 스티븐 호킹 《시간의 역사》 | 6.6% | "가장 많이 구매되고 읽히지 않는 책" |
| 수잔 콜린스 《캣칭 파이어》 | 43.4% | 중반까지 읽음 |
| 도나 타트 《황금방울새》 | **98.5%** | 끝까지 읽음 |

**정확한 패턴**: 비문학(자기계발·인문)은 **앞부분(1~7%)에 집중**, 인기 소설은 **전체에 고르게 분포**. "후반 집중"은 특정 문학 소설에만 해당.

**밑줄 서비스 함의**: 독자가 끝까지 읽은 책일수록 밑줄이 많고 공유율도 높다. 완독률이 높은 에세이·소설 독자가 핵심 타겟.

## 12. 소셜 공유 포맷 최적 데이터 (2024–2025)

### 인스타그램 참여율

| 포맷 | 평균 참여율 |
|------|-----------|
| 캐러셀 (다중 인용) | ~0.55%, 저장률 22% 높음 |
| 단일 이미지 인용 | ~0.35% |
| 릴스 + 인용 오버레이 | 6~7% |

인스타그램 알고리즘은 **저장·DM 공유를 좋아요보다 3배 높게** 가중치. 밑줄 카드는 저장 유도형으로 설계해야 함.

### 최적 인용 길이

| 플랫폼 | 최적 길이 |
|--------|---------|
| BookTok | **8~11단어** |
| Twitter/X | 71~100자 (17% 높은 참여율) |
| 인스타그램 | 캐러셀 형태 (여러 문장 묶음) |

### 바이럴 공식 (Berger & Milkman, Wharton)

바이럴을 결정하는 것은 **감정의 방향(긍정/부정)이 아니라 각성 수준(arousal)**이다.

| 감정 유형 | 각성 수준 | 바이럴 예측 |
|----------|---------|-----------|
| 경이·경탄 | 높음 | **최고** |
| 그리움·사랑 | 높음 | 매우 높음 |
| 분노·불의 | 높음 | 높음 |
| 실용적 지혜 | 중간 | 중간 |
| 철학적 추상 | 낮음 | 낮음 |
| 슬픔·우울 | **낮음** | **최저** |

→ "슬프지만 사실인 문장"이 한국 북스타그램에서 인기 있어도, 순수 슬픔 문장은 국제적으로 바이럴 약함. **슬픔을 경이로 재프레이밍한 문장**이 가장 강력.

### 바이럴 문장 3가지 구조

1. **Double-Truth Reveal** — 두 번째 문장이 첫 번째를 감정적으로 재프레이밍
2. **Universal Specific** — 극도로 구체적인 디테일이 역설적으로 보편성을 만듦
3. **Lyrical Validation** — 독자가 이미 느끼지만 말하지 못한 상태를 정확히 표현

## 13. 밑줄 = 정체성 선언 (학술 종합)

- **Jackson (2001, Yale)**: 여백 주석은 "이 책을 읽었고, 사랑했고, 참여했다는 선언"
- **Barnett (2014)**: Kindle 하이라이트는 자기계발·심리·성장 텍스트에 집중 — 정체성 공명 콘텐츠
- **Dera (2024, BookTok 연구)**: "bookish identity"가 현대인의 자아 개념으로 등장. 밑줄 공유 = 정체성 구성 행위
- **IJRISS 체계적 리뷰**: 주석은 "시간 캡슐" — 그 순간 나는 어떤 사람이었는지의 기록

**결론**: 밑줄은 메모가 아니라 **자기 선언**이다. 공유는 타인에게 "나는 이런 문장에 멈추는 사람"이라고 말하는 행위.
