## 2. 서비스 본질 & 핵심 메시지

### 서비스 본질 (모든 메시지의 뿌리)

> **당신이 멈춘 문장, 누군가도 거기서 멈췄다.**

독자가 책을 읽다 멈추는 순간에는 이유가 있다.
편집자가 고른 명언이 아니라, 실제 독자들이 멈춘 문장 — **누가 멈췄는지가 그 문장의 맥락이 된다.**

이게 굿리즈, 명언 앱, Readwise와 다른 지점이다.
그들은 문장을 모은다. 밑줄은 문장에 독자의 시선을 붙인다.

### 사업계획서 전체에서 반복할 메시지 3가지

1. **"이미 쓰고 있다"** — Claude Vision API 프로덕션 운영 중. 리스크 없는 확장.
2. **"아무도 안 한다"** — 종이책+소셜+한국어+무료 조합. 특히 "독자의 시선이 문장의 맥락이 되는" 구조는 세상에 없다.
3. **"시장이 이미 존재한다"** — 독자들은 이미 인스타에 밑줄 사진을 올린다. 행동은 있고 인프라가 없다.

## 4. 수익 모델

### 프리미엄 구독 구조

| | 무료 플랜 | 프리미엄 플랜 |
|-|----------|-------------|
| 월 업로드 | **10개 제한** | 무제한 |
| 소셜 피드·좋아요 | ✓ | ✓ |
| 공유 카드 | 기본 3종 | 6종 전체 + 워터마크 제거 |
| **인용 서비스** (출처 포함 텍스트) | ✗ | ✓ |
| AI 취향 추천 피드 | ✗ | ✓ |

> **월 10개 제한 전략**: 월 2~3권 독자는 무료로 충분 → 부담 없이 유입.
> 월 4권+ 또는 크리에이터·작가는 자연스럽게 한도 도달 → 유료 전환.
> "강제"가 아닌 "더 쓰고 싶어서" 결제하는 구조.

### 수익 레이어 전체

| 단계 | 수익원 | 대상 | 시점 |
|------|--------|------|------|
| 즉시 | 도서 어필리에이트 (쿠팡파트너스, 알라딘) | B2C | 6개월 내 |
| 단기 | 인용 서비스 구독 (월 4,900원) | B2C 크리에이터·작가·방송작가 | 6개월 내 |
| 중기 | 저자·출판사 마케팅 데이터 리포트 | B2B | 12개월 내 |
| 장기 | AI 취향 기반 책 큐레이션 구독 | B2C | 18개월 내 |

> **인용 서비스 핵심**: 콘텐츠 크리에이터, 뉴스레터 작성자, 방송 작가, 광고 기획자가
> 책 문장을 인용할 때 필요한 것은 단순히 좋은 문장이 아니다.
> **"내 청중과 비슷한 독자들이 반응한 문장"** — 출처(책명, 저자, 출판사, 페이지) + 어떤 독자들이 멈췄는지 맥락까지 제공.
> 이게 일반 명언 앱과의 결정적 차이.

> **출판사 B2B**: "이 책에서 독자들이 가장 많이 밑줄 친 문장 TOP10" 리포트.
> 저자 대시보드 — 어떤 페이지에서 독자들이 멈췄는지 → 마케팅 카피 소스.

## 5. 마케팅 전략

### 핵심 채널 (우선순위 순)

| 채널 | 방식 | 왜 효과적인가 |
|------|------|--------------|
| **저자** | 저자에게 "내 책 밑줄 데이터" 무료 제공 → SNS 홍보 유도 | 저자가 공유하면 팬덤 자동 유입 + 출판사 B2B 연결 |
| **북클럽·독서 모임** | 모임 내 무료 체험 → 밑줄 공유 → 피드 형성 | 이미 책 얘기하는 사람 집합소. 전환율 최고 |
| **북스타그래머** | 사용 경험 기반 콘텐츠 협업 (광고 X) | "써봤더니 좋더라" 진정성이 팔로워 신뢰로 이어짐 |
| **오픈카톡방 독서 모임** | 서비스 소개 + 밑줄 카드 예시 공유 | 타겟 집결, 비용 0 |
| **어필리에이트** | 밑줄 상세 → "이 책 구매하기" (쿠팡, 알라딘) | 책에 관심 있는 순간에 노출 |

### 마케팅 예산 800만원

| 항목 | 금액 | 내용 |
|------|------|------|
| 저자 파트너십 | 200만원 | 저자 10~20명 대시보드 무료 제공 + 공유 요청 |
| 북스타그래머 체험 협업 | 250만원 | 팔로워 1만~5만 계정 5명, 체험비 |
| 독서 모임 파트너십 | 150만원 | 주요 북클럽 3~5곳 이벤트 비용 |
| 콘텐츠 제작 | 100만원 | 카드뉴스, 릴스, 사용 가이드 |
| SNS 광고 | 100만원 | 인스타 북스타그램 관심사 타겟 |

### 바이럴 루프

```
독자 사진 촬영 → Claude AI → 밑줄 + 책 자동 인식 → 저장
        ↓
"같은 문장에서 멈춘 사람들" 알림 → 공명 경험
        ↓
공유 카드 생성 (6종 테마, "당신이 멈춘 문장" 카피 포함)
        ↓
인스타 스토리/피드 업로드 → 앱 링크 자동 포함
        ↓
팔로워 방문 → "나도 이 문장에서 멈췄는데" → 신규 가입 (반복)
```

> 바이럴의 핵심은 카드 디자인이 아니라 **"나도 여기서 멈췄다"는 공명**이다.
> 같은 문장에서 멈춘 사람이 이미 있다는 사실이 클릭을 만든다.
