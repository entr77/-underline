# 밑줄 용어사전 (Brand Glossary)

> 모든 화면 텍스트는 이 용어사전을 기준으로 작성한다.
> 카피라이팅/기획/마케팅/개발 전 직군 공통 기준.

---

## 브랜드 목소리 (Brand Voice)

- **톤**: 따뜻하고 시적. 독자에게 말 걸 듯 자연스럽게.
- **시제**: 현재형 중심 ("저장됐어요" 아니라 "남았어요")
- **인칭**: 2인칭 "당신" 또는 1인칭 "나" — 앱이 사용자와 대화하는 느낌
- **피하는 것**: 기술적·행정적 단어 (기록, 저장, 업로드, 처리, 등록, 관리)

## 핵심 은유 (Core Metaphors)

| 은유 | 뜻 | 쓰임 |
|------|-----|------|
| **멈추다** | 문장에 마음이 걸리는 순간 | "당신이 멈춘 문장", "같이 멈춘 사람" |
| **남기다** | 밑줄을 기록하는 행위 | "문장을 남기다", "밑줄을 남기다" |
| **펼치다** | 책을 펼치는 행위 | "펼친 책", "처음 펼치는 날" |
| **긋다** | 밑줄을 그리는 물리적 행위 | "처음 줄을 긋는 날" |

## 핵심 명사 (Core Nouns)

| 개념 | 공식 용어 | 쓰지 않는 표현 |
|------|----------|--------------|
| 앱 이름 | **밑줄** | Underline |
| 밑줄 친 문장 하나 | **밑줄**, **문장** | 기록, 포스트, 항목 |
| 밑줄을 남기는 행위 | **밑줄 긋기**, **남기기** | 기록하기, 저장하기, 업로드 |
| 밑줄 모음 화면 | **피드** | 타임라인, 홈 |
| 책별 밑줄 모음 | **책** (탭) | 도서관, 서재 |
| 내 밑줄 모음 | **내 프로필** | 마이페이지 |
| 좋아요 | **같이 멈춘** (명사), **공감** (동사) | 좋아요, 하트, 리액션 |
| 팔로워 | **함께 읽는 사람** | 팔로워, 구독자 |

| 스텝 | 타이틀 | 스텝 인디케이터 |
|------|--------|----------------|
| 사진 업로드 | 밑줄 남기기 | 사진 |

#### 처리 중 메시지 (순환)
1. "페이지 위의 문자들을 읽고 있어요"
2. "당신이 멈춘 자리를 찾고 있어요"
3. "이 문장이 어느 책에서 왔는지 확인해요"
4. "밑줄 위의 감정을 읽고 있어요"

#### 버튼 텍스트
| 현재 | 공식 |
|------|------|
| 이 문장 저장하기 | **이 문장 남기기** |

## 에러 / 안내 메시지 규칙

- 에러: "~에 실패했어요" X → "~이 안 됐어요" O (부드럽게)
- 안내: 명령형 X → 제안형 O ("하세요" → "해보세요" / "해요")
- 빈 상태: 항상 2줄 — 상황 설명 + 행동 유도
