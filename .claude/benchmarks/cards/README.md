## ref-01 — 달리기 책 카드 (인스타그램 게시물)

**레이아웃**
- 흰 배경 (크림/화이트)
- 상단: 책 표지 이미지 — 중앙 정렬, 그림자 없이 클린하게
- 중단: 본문 설명 텍스트 — 세리프, 중앙 정렬, 2~3줄
- 하단: 책 제목을 guillemet 기호로 강조 (`《 》`) — 굵은 세리프, 크게

**타이포그래피**
- 설명 텍스트: 세리프 계열, 16~18px, line-height 넉넉, 중앙 정렬
- 제목 강조: `《 그저 달리기인 줄 알았는데 삶이었다 》` — 볼드, 20px+
- guillemet(`«»`, `《》`) 사용 — 따옴표 대신 서적 강조에 효과적

**느낌**
- 출판사 공식 SNS 스타일
- 텍스트가 주인공, 이미지는 보조
- 여백 넉넉 → 고급스러움

## ref-02 — 혁신의 지리학 카드 (인스타그램 게시물)

**레이아웃**
- 흰 배경
- 상단: 책 제목 (`『혁신의 지리학』`) — 소형 텍스트, 좌측 정렬 또는 중앙
- 중단: 태블릿에 책 표지 띄운 목업 이미지 — 입체감 강조
- 하단: 핵심 문구 — 세리프, 중앙 정렬, 2줄
- 좌하단: 작은 프로필 아바타 (원형)

**타이포그래피**
- 제목 브래킷: `『 』` — 한국 서적 인용 기호
- 본문: 세리프, 크림색 배경 위 다크 잉크 색상
- 질문형 문구: "왜 어떤 곳에서는 ... 못한가" — 독자 참여 유도

**느낌**
- 서점/독서 커뮤니티 게시 스타일
- 목업 이미지로 실물감 강조
- 책 제목을 카드 상단에 작게 표기 → 맥락 먼저 제공

## ref-03 — 환상의 빛 카드

**레이아웃**
- 배경: 책 표지 이미지를 블러 처리한 그레이 톤
- 상단: 책 표지 이미지 — 크게, 중앙 정렬 (카드 면적의 ~50%)
- 중단: 책 제목 `『환상의 빛』` — 굵은 세리프, 중앙
- 하단: 긴 인용문 (3~4줄) — 세리프, `" "` 따옴표 포함, 중앙 정렬
- 좌하단: 작은 원형 아바타

**특징**
- 배경색이 책 표지 무드를 따라감 (흑백 → 회색 배경)
- 인용문이 길어도 여백 없이 꽉 채움
- 전체적으로 문학적, 무거운 분위기

## ref-04 — 당신 인생의 이야기 카드

**레이아웃**
- 배경: 짙은 네이비/블루 그라디언트
- 상단: 책 표지 — 중앙, 크게
- 중단: `『당신 인생의 이야기』` — 흰색 굵은 세리프, 크게
- 하단: 인용문 2단락 — 세리프, 흰색, `" "` 따옴표
- 좌하단: 원형 아바타

**특징**
- 배경 그라디언트가 책 표지 색감(파랑/하늘)을 그대로 확장
- 제목이 매우 크고 강함 → 책 이름이 먼저 눈에 들어옴
- SF 장르 무드를 배경색으로 표현

## ref-05 — 휴먼카인드 카드

**레이아웃**
- 배경: 책 표지 타이포그래피를 블러한 다크 배경
- 상단: 책 표지 (앞/뒤 합친 입체감 있는 이미지) — 중앙
- 중단: `『휴먼카인드』` — 흰색 굵은 세리프
- 하단: 긴 인용문 (4줄+) — 세리프, `" "` 따옴표, 중앙 정렬
- 좌하단: 원형 아바타

**특징**
- 배경이 책의 내용(텍스트)을 블러해서 사용 — 매우 독특
- 인용문이 가장 길고 무거움 → 콘텐츠 밀도 높음
- 전체 카드가 어두운 계열 → 진지한 논픽션 느낌

## 밑줄 카드에 적용 가능한 인사이트

| 요소 | 참고 사례 | 밑줄 적용 아이디어 |
|------|---------|-----------------|
| guillemet 기호 | `《 》`, `『 』` | 책 제목 표시에 활용 가능 |
| 흰 배경 카드 변형 | ref-01, ref-02 | 라이트 테마 card_bg 옵션 추가 |
| 텍스트가 주인공 | ref-01 | 이미지 없이 텍스트만으로 완성도 있는 카드 |
| 프로필 아바타 위치 | ref-02~05 (좌하단 소형) | 카드 외부로 뺀 현재 구조와 유사 |
| 여백 | ref-01, ref-02 | 카드 내 padding 5 이상 유지 |
| 배경 = 책 무드 확장 | ref-03~05 | 블러 배경 + 그라디언트로 책 분위기 표현 |
| 긴 인용문 | ref-03~05 | 4줄 이상도 읽힘 — line-height 충분히 |
| `" "` 따옴표 인용 | ref-03~05 | 인용문 앞뒤 따옴표 스타일 옵션 고려 |

## ref-06 — 밑줄 애니메이션 카드 (Instagram Reel)

**출처**: https://www.instagram.com/reel/DZefQjHx2QQ/

**레이아웃/애니메이션 아이디어**
- 텍스트가 먼저 나타나고, 밑줄이 왼쪽→오른쪽으로 그어지는 애니메이션
- 마치 실제로 형광펜을 긋는 것처럼 `scaleX` 트랜지션

**밑줄 앱 구현 아이디어**
```
카드 진입 시:
1. 텍스트 fade-in (0.3s)
2. 노란 하이라이트 바가 왼쪽에서 오른쪽으로 확장 (0.4s, ease-out)
   → CSS: transform: scaleX(0→1), transform-origin: left

공유 이미지 카드(PNG)에는 정적으로,
피드 카드(웹)에는 IntersectionObserver로 뷰포트 진입 시 1회 실행
```

**구현 포인트**
- 현재 노란 바(`w-7 h-[3px]`)를 가로로 더 길게 (텍스트 너비만큼)
- `animate-[underline-draw_0.4s_ease-out_0.3s_both]` 커스텀 keyframe 추가
- Tailwind `@keyframes underline-draw { from { transform: scaleX(0) } to { transform: scaleX(1) } }`
